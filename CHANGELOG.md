# Changelog

## [Unreleased]

## [1.1.0] - 2026-02-18

### Added

- `融合度` パラメーターを追加。
- 翻訳ファイル（英語）を追加。

### Changed

- 一部パラメーターをグループ化。
- 回転に関するパラメーターの下限と上限を `-180`、`180` から `-360`、`360` に変更。
- `形状` の `星形` を `星型` に変更。
- ファイル名を `カラーハーフトーン.anm2` から `ColorHalftone.anm2` に変更。

## [1.0.0] - 2025-11-28

- 初版。
