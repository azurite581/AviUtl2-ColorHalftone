# Changelog

## [Unreleased]

## [2.0.0] - 2026-02-25

### Added

- 背景の色を変更できるようにした（`背景色`）。
- 背景の透明度を変更できるようにした（`背景色透明度`）。
- 元のオブジェクトのアルファを維持するどうかを選択できるようにした（`元のアルファを維持`）。
- 混色方法を選択できるようにした（`混色方法`）。選択項目として `減法混色`、`加法混色` を追加。
- 背景に関連する設定を `背景` としてグループ化。

### Changed

- `サイズ` の下限を `1` に変更。
- 機能追加に伴い `PI` の項目の順番を変更。
- グループ化した項目はデフォルトで折りたたむようにした。
- 設定項目追加に伴い翻訳ファイルを更新。

### Removed

- `背景色透過` のチェックボックスを削除。代わりに背景の透明度をトラックバーから指定できるようにした。

### Fixed

- スクリプトに記載したバージョン情報が正しくなかったのを修正。

## [1.1.1] - 2026-02-18

### Fixed

- ドットがセル境界を超えて描画される際に正しく描画されない問題を修正。

## [1.1.0] - 2026-02-18

### Added

- `融合度` パラメーターを追加。
- 翻訳ファイル（英語）を追加。

### Changed

- 一部パラメーターをグループ化。
- 回転に関するパラメーターの下限と上限を `-180`、`180` から `-360`、`360` に変更。
- `形状` の `星形` を `星型` に変更。
- ファイル名を `カラーハーフトーン.anm2` から `ColorHalftone.anm2` に変更。

## [1.0.0] - 2025-11-28

- 初版。
